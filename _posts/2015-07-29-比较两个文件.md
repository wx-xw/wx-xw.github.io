#比较两个文件

文件1

```awk
a
b
c
d
e
```

文件2

```awk
1
2
3
a
b
c
```

###grep命令
grep命令主要做的是留下不包含文件1中所有数据的文件2的数据

```awk
grep -vFf 文件1 文件2
# v 不包含
# F 固定字符串，即为全匹配
# f 文件
### output ###
# 1
# 2
# 3
```

###comm命令
```awk
comm 文件1 文件2

### output ###
### 第一列为文件1独有的内容，第二列为文件2独有的内容，第三列为共有的内容 ###
# 	1
# 	2
# 	3
# 		a
# 		b
# 		c
# d
# e

comm -1 文件1 文件2
### output ###
### -1 参数为去除文件1独有内容，-2 参数为去除文件2独有内容，-3 参数为去除共有内容。三个参数可以共同使用 ###
# 1
# 2
# 3
# 	a
# 	b
# 	c

comm -12 /tmp/tmp1 /tmp/tmp2
### output ###
# a
# b
# c

```
